#이 프로젝트는 3ds Max 바이패드로 리깅된 캐릭터와 모션캡쳐 애니메이션을 포함하고 있습니다. 

##유니티 6000.3.5 버전에서 테스트되었습니다.

모션캡쳐 fbx는 무슨 이유인지 깃허브에 올라가지 않아서 제외했습니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| 작업 인원 / 기간 | 1명/ 2주 |
| 작업 소프트웨어 | 3ds Max 2025, Unity 6 | 
| 리깅 방식 | Biped (Bip001) | 
| 주요 기능 | MotionCapture 로코코 , Skin (헤어/몸/옷) 3dsmax | 
| 애니메이션 | 기타모션 및 드럼 모션을 유니티 엔진에 애니메이션컨트롤러를 사용해서 적용해본다. | 
| 성과 or 문제해결| 첫번째 프로젝트의 아쉬움을 느껴 스스로 프로젝트를 완수해보고자 시작했습니다. 3dsmax와 유니티 툰 세이더에 대해 공부를 하여 적용해 볼 수 있었습니다. 스킨 작업에 대해 처음 시도해보았는데, 버텍스에 값을 할당할 때 시행착오를 겪었습니다. 버텍스 선택에서 Ctrl 키로 하나 하나 잡다가, roop 기능을 사용해서 한줄을 한번에 잡는다던가, glow 기능을 사용해서 여러 버텍스를 잡기 편해지는것이 나중에 알게 되었습니다. |
